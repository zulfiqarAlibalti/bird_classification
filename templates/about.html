About This Project

This project is a bird classification application built using Flask, PyTorch, and React. The application allows users to upload images of birds, which are then processed by a pre-trained neural network to predict the species of the bird.

The project showcases the integration of machine learning models into a web application, providing a user-friendly interface for users to interact with the model.

For more details, visit the GitHub repository.
